### Skeleton items

We can use `PfSkeleton` as the `component` prop of each `pf-gallery-item` to create a skeleton layout.

<Canvas withSource="open">
  <Story
    name="Skeleton items"
    args={{
      component: PfSkeleton,
    }}
    decorators={[
      () => ({
        components: { PfGallery, PfGalleryItem },
        setup() {
          return {
            component: markRaw(PfSkeleton),
          };
        },
        template: `
          <!-- storybook wants to transform the root element to a skeleton itself, so we give it a div to play with -->
          <div>
            <pf-gallery gutter>
              <story style="border-color: var(--pf-global--palette--green-300)" />
              <pf-gallery-item :component="component" />
              <pf-gallery-item :component="component" />
              <pf-gallery-item :component="component" />
              <pf-gallery-item :component="component" />
              <pf-gallery-item :component="component" />
            </pf-gallery>
          </div>
        `,
      }),
    ]}>
    {Template.bind({})}
  </Story>
</Canvas>
