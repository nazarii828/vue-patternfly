# PfStackItem `pf-stack-item`

## Examples

### Basic

The dashed lines are not part of the component and are only for visual separation in the documentation. The highlighted item with green lines is the one controlled by the story.

<Canvas withSource="open">
  <Story
    name="Basic"
    args={{
      filled: true,

    decorators={[
      () => ({
        components: { PfStack, PfStackItem },
        template: `
          <pf-stack gutter class="docs-layout-outline" style="height: 300px">
            <pf-stack-item>content</pf-stack-item>
            <story style="border-color: var(--pf-global--palette--green-300)" />
            <pf-stack-item>content</pf-stack-item>
          </pf-stack>
        `,
      }),
    ]}>
    {Template.bind({})}

### Skeleton items

We can use `PfSkeleton` as the `component` prop of each `pf-stack-item` to create a skeleton layout.

<Canvas withSource="open">
  <Story
    name="Skeleton items"
    args={{
      filled: true,
      component: PfSkeleton,
    }}
    decorators={[
      () => ({
        components: { PfStack, PfStackItem },
        setup() {
          return {
            component: markRaw(PfSkeleton),
          };
        },
        template: `
          <!-- storybook wants to transform the root element to a skeleton itself, so we give it a div to play with -->
          <div>
            <pf-stack gutter style="height: 300px">
              <pf-stack-item :component="component" />
              <story style="border-color: var(--pf-global--palette--green-300)" />
              <pf-stack-item :component="component" />
            </pf-stack>
          </div>
        `,
      }),
    ]}>
    {Template.bind({})}
  </Story>
</Canvas>
